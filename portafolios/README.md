# Portafolio Profesional - Luis Colado Jiménez

## 📋 Descripción del Proyecto

Portafolio web profesional de una sola página (single-page) para **Luis Colado Jiménez**, Técnico Electrónico, Informático y Electricista con amplia experiencia multidisciplinar.

## 🎯 Características Implementadas

### ✅ Funcionalidades Completadas

1. **Navegación Responsiva**
   - Menú de navegación fijo con scroll suave
   - Menú hamburguesa para dispositivos móviles
   - Navegación activa según la sección visible
   - Animaciones y transiciones fluidas

2. **Sección Hero (Inicio)**
   - Presentación profesional con gradiente moderno
   - Enlaces a redes sociales (LinkedIn, Email, Teléfono)
   - Botones de llamada a la acción
   - Efecto de escritura animado en el título
   - Diseño responsivo con imagen de perfil

3. **Sección Sobre Mí**
   - Información personal y profesional en tarjetas
   - Datos de contacto y ubicación
   - Idiomas, disponibilidad y movilidad
   - Diseño en grid responsivo

4. **Sección Experiencia Profesional**
   - Tres categorías de experiencia:
     - **Electrónica y Tecnología**: 8 posiciones (2009-2025)
     - **Informática y Desarrollo**: 4 posiciones (2009-2024)
     - **Electricidad y Mantenimiento**: 4 posiciones (2023-2025)
   - Timeline visual con línea de tiempo
   - Animaciones al hacer scroll
   - Información detallada de cada puesto

5. **Sección Educación y Certificaciones**
   - 3 títulos académicos destacados:
     - Grado Superior en Desarrollo de Aplicaciones Web (2023)
     - Certificado Profesional ELEE0109 (2024)
     - Grado Medio en Electrónica de Consumo (2009)
   - 8 certificaciones profesionales adicionales
   - Diseño en tarjetas con iconos Font Awesome

6. **Sección Habilidades**
   - Habilidades técnicas con barras de progreso animadas
   - Software y herramientas (7 tecnologías)
   - Competencias profesionales destacadas
   - Animación al entrar en la vista

7. **Sección Contacto**
   - Formulario de contacto funcional (mailto)
   - Información de contacto completa
   - Badge de disponibilidad inmediata
   - Diseño atractivo con fondo degradado

8. **Elementos Adicionales**
   - Botón "Scroll to Top" con animación
   - Footer con enlaces sociales
   - Animaciones al hacer scroll (Intersection Observer)
   - Diseño 100% responsivo para todos los dispositivos

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos modernos con variables CSS, flexbox y grid
- **JavaScript (Vanilla)**: Interactividad sin frameworks
- **Font Awesome 6.4.0**: Iconos profesionales
- **Google Fonts (Poppins)**: Tipografía moderna
